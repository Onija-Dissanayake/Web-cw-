Role: Shared Template & Navigation, Home Page, Gallery

Home Page

Technical Description

The Home page serves as the entry point for the "Life on Land" project. I utilized a 4-column CSS Grid layout to create the visual navigation at the bottom of the page. Flexbox was used in the header and for the central information cards to ensure a responsive and centered layout. I also implemented CSS transitions to add a scale effect to the images upon hover, improving user engagement.

Accessibility

I ensured that all images have descriptive alt text and used a hierarchical heading structure (h1-h3) to assist screen readers. The color palette was checked to ensure sufficient contrast between the text and background for readability.

Gallery

Technical Description

The Gallery uses a 2-column grid layout. I implemented an interactive modal system using JavaScript. By utilizing HTML5 'data-' attributes, I allowed the script to dynamically swap content into the modal without page reloads. I also added a user customization feature that allows visitors to change the modal's background color and font family using DOM manipulation.

Accessibility

The modal includes a clear close button and is designed to close if the user clicks outside the modal area. All thumbnails include alt text to describe the wildlife and land-based ecosystems displayed.

Content Page Student 1

Technical Description

I built the Content page to present educational data on the importance of biodiversity. I implemented internal navigation using anchor links (href="#id") for ease of access. I also used CSS 'position: fixed' for a "Go to Top" button to help users navigate long sections of text quickly.

Accessibility

The page follows a strict semantic structure. The internal navigation list serves as a skip-link alternative for keyboard-only users, allowing them to bypass content and go straight to specific sections.

Challenges and Lessons Learned

The primary challenge was coordinating the global style requirements with the team. earning to use Git to pull updates and resolve merge conflicts in VS Code was a vital part of the process. Technically, mastering the z-index and positioning for the interactive gallery modal was the most difficult task, as it required precise control over the visual layers of the page.

Compliance

All content and code within this project are compliant with JANET regulations. The website is strictly for academic and educational purposes related to UN Sustainable Development Goal 15. All external media assets were sourced from royalty-free platforms (Unsplash) and are properly referenced.
