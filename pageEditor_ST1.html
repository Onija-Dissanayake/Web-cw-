2. Technical Description of Implemented Pages

Shared Template & Global CSS

HTML Structure: Established the baseline semantic HTML framework using <header>, <nav>, <main>, and <footer> to ensure a consistent layout across the site.

CSS Techniques: Created global.css using a universal selector reset and set the typography to "Poppins". Utilised CSS Flexbox in the header and navigation for horizontal alignment and spacing. Implemented a linear gradient background and glassmorphism effects (using backdrop-filter: blur and rgba backgrounds) for the cards and UI elements.

Design Rationale: The dark green and gold (#c7a46a) colour palette was chosen to reflect the "Life on Land" nature theme while maintaining high contrast for readability.

Home Page

View Live Home Page ↗ View Validation →

HTML Structure: Utilised structured <section> tags to separate the intro, information cards, and the visual gallery links.

CSS Techniques: Used embedded CSS to style page-specific elements without conflicting with the global stylesheet. Deployed CSS Grid (grid-template-columns: repeat(4, 1fr)) for the bottom navigation gallery and Flexbox for the central information cards. Added scale transforms on hover to create an interactive feel.

Interactive Gallery

View Live Gallery Page ↗ View Validation →

HTML Structure: Used data- attributes within the gallery items to store dynamic content (titles, descriptions, image paths) which are later retrieved by JavaScript.

CSS & JS Techniques: Built a responsive 2-column grid. The interactive modal was built using a hidden div that toggles a .show class via JavaScript event listeners. JavaScript was also used to fulfill the user customisation requirement, allowing the user to dynamically alter the modal's background colour and font family via DOM manipulation.

Content Page

View Live Content Page ↗ View Validation →

HTML Structure: Heavily relied on semantic headings (<h1> to <h3>) to create a logical document outline detailing the importance of biodiversity.

Implementation: Implemented internal page navigation using anchor links (href="#id") pointing to specific sections. Added a fixed position "Go to Top" button to enhance user experience on long scrolls.

3. Challenges & Lessons Learned

One of the main technical challenges was implementing the interactive JavaScript modal on the Gallery page without disrupting the global CSS layout. Initially, the modal overlay was blocking clicks on the main page even when hidden. I resolved this by using a combination of opacity: 0 and visibility: hidden in CSS, toggled via JavaScript classes.

This process taught me the importance of isolating component states in CSS and how powerful data- attributes can be for passing HTML information directly into JavaScript variables.

4. Accessibility Considerations

  • Semantic HTML: Used landmarks like <main> and <nav> to help screen readers navigate the page structure easily.
  • Alt Text: Ensured every <img> tag includes descriptive alt text (e.g., "Forest representing life on land") for visually impaired users.
  • Visual Contrast: Selected text colours (white and gold against dark green) that provide high contrast to make reading easier.
